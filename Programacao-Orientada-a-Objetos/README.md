# 📚 Programação Orientada a Objetos

![Java](https://go-skill-icons.vercel.app/api/icons?i=java&theme=light) ![Eclipse](https://go-skill-icons.vercel.app/api/icons?i=eclipse&theme=light) ![IntelliJ](https://go-skill-icons.vercel.app/api/icons?i=idea&theme=light)

## 📖 Descrição

Esta disciplina introduz os conceitos fundamentais da Programação Orientada a Objetos (POO) utilizando a linguagem Java. O foco está na aplicação dos pilares da orientação a objetos para criar código mais organizado, reutilizável e de fácil manutenção.

## 🎯 Objetivos de Aprendizagem

- Compreender e aplicar os **4 pilares da POO**
- Desenvolver aplicações em **Java** seguindo boas práticas
- Implementar **estruturas de dados orientadas a objetos**
- Aplicar **padrões de design** fundamentais
- Criar **interfaces gráficas** básicas

## 🏗️ Pilares da Orientação a Objetos

### 🔒 Encapsulamento

- Proteção de dados através de modificadores de acesso
- Métodos getters e setters
- Princípio da ocultação de informações

### 🧬 Herança

- Reutilização de código através de hierarquias
- Classes base e derivadas
- Modificador `extends` e `super`

### 🎭 Polimorfismo

- Capacidade de objetos assumirem múltiplas formas
- Sobrescrita e sobrecarga de métodos
- Interfaces e classes abstratas

### 📊 Abstração

- Criação de modelos simplificados da realidade
- Classes abstratas e interfaces
- Foco nos aspectos essenciais

## 🚀 Como Compilar e Executar

### Pré-requisitos

```bash
# Verificar instalação do Java
java -version
javac -version
```

### Compilação

```bash
# Compilar arquivo individual
javac NomeClasse.java

# Compilar projeto completo
javac -cp . *.java
```

### Execução

```bash
# Executar aplicação
java NomeClasse

# Executar com classpath específico
java -cp . pacote.NomeClasse
```

## 📚 Conceitos Fundamentais Abordados

- **Classes e Objetos:** Definição e instanciação
- **Atributos e Métodos:** Variáveis de instância e comportamentos
- **Construtores:** Inicialização de objetos
- **Modificadores de Acesso:** public, private, protected
- **Herança:** Especialização e generalização
- **Polimorfismo:** Flexibilidade e extensibilidade
- **Interfaces:** Contratos de comportamento
- **Tratamento de Exceções:** Robustez de aplicações

## 📈 Progressão de Aprendizado

1. **Básico:** Classes, objetos, atributos, métodos
2. **Intermediário:** Herança, encapsulamento, construtores
3. **Avançado:** Polimorfismo, interfaces, classes abstratas
4. **Especializado:** Padrões de design, tratamento de exceções
